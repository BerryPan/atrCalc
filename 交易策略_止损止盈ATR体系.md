# 交易策略：止损 / 止盈 / ATR 体系

> 来源：微信 ClawBot，整理于 2026-06-17

---

## 一、止损

| 参数 | 说明 |
|------|------|
| **触发比例** | max(7%, 2×入场ATR) |
| **触发价** | 成本 × (1 − 触发比例) |
| **入场ATR** | 买入日前14个交易日逐日TR%的均值，**首次计算后锁定不变** |

---

## 二、止盈（两阶段）

### 第一阶段 — 减仓 50%

| 参数 | 说明 |
|------|------|
| **条件** | 盘中最高价浮盈 ≥ 5×当前ATR |
| **操作** | 卖出一半仓位 |
| **触发价** | 成本 × (1 + 5×当前ATR) |
| **当前ATR** | 最近14个交易日逐日TR%均值，**每日刷新** |

### 第二阶段 — 余仓清仓

| 参数 | 说明 |
|------|------|
| **条件** | 剩余仓位触发移动止盈 |
| **触发价** | 盘中最高价 × (1 − 2×当前ATR) |
| **峰值** | 止盈触发后的盘中最高价（非收盘价），**每日更新** |

---

## 三、ATR 计算

- **周期**：14日
- **分母**：前日收盘价（**绝不用当日收盘价**）
- **TR 公式**：
  ```
  TR = max(当日高−当日低, |当日高−前收|, |当日低−前收|)
  ```
- **ATR%**：14日 TR% 均值

---

## 四、仓位上限

- **公式**：`0.7% / (2 × 入场ATR%)`
- **示例**：
  - ATR 3.5% → 10% 仓
  - ATR 7% → 5% 仓

---

## 五、做T与减仓规则

- **做T**：同日买卖对冲不影响成本，利润进现金；净多买算加仓，净多卖算减仓
- **减仓**：剩余仓位保持原始成本 + 原始入场ATR，**券商摊薄成本不采纳**
